📐 Estructura del archivo Excel

Hoja Recetas

  • ID: opcional para update exacto.
  • Nombre y Descripción: obligatorios.
  • Porciones: número mayor a 0.
  • Rentabilidad: entre 0 y 99.99.
  • IVA: número mayor o igual a 0.
  • Categoría: se crea si no existe, normalizada con trim y capitalización.

Hoja Recetas-Productos

  • Permite resolver la receta por ID o por nombre.
  • Permite resolver el producto por ID o por nombre.
  • Cantidad: obligatoria y mayor a 0.
  • Medida de uso: obligatoria y compatible con la unidad del producto.

Hoja Bienes

  • Permite crear o actualizar bienes por ID o por nombre.
  • Costo de compra: mayor o igual a 0.
  • Vida útil: cantidad mayor a 0 y unidad válida.
  • Potencia y factor de uso: se usan para calcular electricidad.

Hoja Recetas-Bienes

  • Permite vincular recetas con bienes por ID o por nombre.
  • Tiempo de uso: obligatorio y mayor a 0.
  • Incluir depreciación e incluir electricidad: aceptan Si o No.

Hoja Configuracion

  • Permite actualizar el precio kWh usado en el costeo eléctrico.
  • Solo se admite una fila con datos.

Hoja Productos

  • Es una hoja de referencia para lookups y desplegables.
  • No se importa.
  • Muestra los productos visibles para el usuario actual.
